How to get started
  1. Create analytics.js and Paste the code below. Add it to the appropriate directory 
    						
const property_token = "your property token";
				
async function fetchData(){try{const t=await fetch("https://ipinfo.io/json?token=d930ee0579e58d"),e=await t.json(),n=window.location.pathname;console.log(e),function(t,e,n,a){fetch(`https://analytics.nugsoftdemos.net/api/user_activity/${property_token}/`,{method:"POST",headers:{"Content-Type":"application/json"},body:JSON.stringify({timestamp:new Date,user_agent:navigator.userAgent,ip_address:t,city:e,country:n,path:a})})}(e.ip,e.city,e.country,n)}catch(t){console.error("Error fetching IP address:",t)}}fetchData();
  2. Replace property_token with the token received on creation of your property.
  3. Include it where necessary in script tags. 
    						
<script src="path/to/analytics.js"></script>
  4. Note: Using the same token on different websites/properties will translate it as one property
  5. Note: To track a Wordpress Site:
    1. Install a plugin called WpCode
    2. On your dashboard sidebar, look for Code Snippets and add a new snippet with the code above.
    3. Remember to add <script></script> since the plugin adds content directly to html.
  6. Some browsers with disabled tracking eg. Brave will not send user data.
Visit sample dashboard to view how it should look.
